.. _install:

Installation
============

Using Pypi
----------

mealy can be installed through Pypi using:

.. code-block:: bash

    $ pip install mealy

Installing locally
------------------

You can also fetch the code and install the package from your local repository.
Again, the preferred way is to use pip.

.. code-block:: bash

    $ git clone https://github.com/dataiku/mealy
    $ cd mealy
    $ pip install -e
